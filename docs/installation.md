# Installation

## Stable release

To install CAPWAP Discovery Fuzzer, run this command in your terminal:

```sh
uv add capwap-discovery-fuzzer
```

Or if you prefer to use `pip`:

```sh
pip install capwap-discovery-fuzzer
```

## From source

The source files for CAPWAP Discovery Fuzzer can be downloaded from the [Github repo](https://github.com/sckrt/capwap_discovery_fuzzer).

You can either clone the public repository:

```sh
git clone git://github.com/sckrt/capwap_discovery_fuzzer
```

Or download the [tarball](https://github.com/sckrt/capwap_discovery_fuzzer/tarball/master):

```sh
curl -OJL https://github.com/sckrt/capwap_discovery_fuzzer/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd capwap_discovery_fuzzer
uv pip install .
```
